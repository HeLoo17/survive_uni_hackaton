# Survive Uni

A simple browser app for the Cursor Mini Hackathon — track assignments, plan study sessions, and coordinate group tasks for the week ahead.

## Features

- **This Week** — calendar-style overview of everything due in the next 7 days
- **Assignments** — add deadlines, mark done, edit, delete
- **Study Sessions** — plan time, location, and friends to study with
- **Group Tasks** — assign teammates and share resource links
- Data persists in your browser via `localStorage` (no backend needed)

## Run locally

Open `index.html` in any modern browser, or serve the folder:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Stack

Plain HTML, CSS, and JavaScript — no build step required.
